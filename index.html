MY THOUGHTS ABOUT CODE

I believe that data analysis is moving towards the democratisation of the data, giving to the end user interactive power over the information. Visualization techniques responsive to the user input will provide interpretations of the different scenarions of the system/ model behaviour thus will enable more comprehensive data-driven decisions.

MORE THOUGHTS

The emergence of the new libraries, such as Streamlit in Python and R-Shiny in R gives to the developer toole to create basic interactive applications by translating code to Javascript. However these options are quite limited. On the other hand direct injection of Javascript into both other programming languages becoming feasible thus opening ways for interesting and synergistic interactions across language platforms.
